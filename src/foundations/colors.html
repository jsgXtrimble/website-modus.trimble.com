Color Palette

The Ether Framework palette is based on the Trimble brand colors. You can modify the colors in _theme.scss and _theme-dark.scss when using the SASS implementation.

Overview

Our framework includes 4 sets of colors:

  1. Theme Colors
  2. UI Colors
  3. Trimble Brand Colors
  4. Gray Colors

Each set of colors has its own set of utility classes that modify background color, font color, and border color. You can reference the available utility classes below:
