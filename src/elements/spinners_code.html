Overview

Our spinners are built entirely using HTML and CSS. You can toggle their visibility using JavaScript in you project. Their appearance, alignment, and sizing can be easily customized with our utility classes.

For accessibility purposes, each loader here includes role="status" and a nested <span class="sr-only">Loading...</span>.

Color Variants

The border spinner uses currentColor for its border-color, meaning you can customize the color with text color utilities.

However, we recommend refraining from using too many different colored spinners. We recommend using our primary, dark, and tertiary colors for your use cases.
