Ether Layout

Besides the aesthetics and UI elements, the Ether Framework has an optional layout you can use for your web applications.

Basic Structure

The basic structure for our Ether Layout, including the sidebar, toolbar, and panel, is as follows: 

  div.ether-layout
        ├── nav.ether-header
        ├── div.ether-body
        |   ├── nav.ether-sidebar
        |   └── div.ether-content-rows
        |       ├── div.ether-toolbar(optional)
        |       └── div.ether-content-columns
        |           ├── div.ether-panel(optional)
        |           └── div.ether-content
        └── div.ether-footer

The page should be wrapped in a <div class="ether-layout"> to set up the overall structure.

The top nav bar is a standard Bootstrap <nav class="navbar navbar-expand-lg"> with an additional class of .ether-header to apply the aesthetics. Please refer to the Bootstrap Navbar Docs for more information.

The .ether-header should be followed by <div class="ether-body"> to set up the left nav area and main content area.

The left nav is a Bootstrap <nav class="nav flex-column"> to create a vertical nav along with an additional class of .ether-sidebar for the proper styling. Within each .navlink, it is important to have an icon or image with a class of .left-nav-icon

The next sibling element after the .ether-sidebar should be .ether-content-rows. An optional child of .ether-toolbar.

After .ether-content-rows should be .ether-content-columns with an optional child of .ether-panel.

Your main content is housed within .ether-content inside of .ether-content-columns.

After the .ether-body, the next element should be <div class="ether-footer">.

Ether Toolbar Example

Our framework includes an all purpose toolbar that can be inserted inside your layout. Simply add .ether-toolbar inside .ether-content-rows and begin adding your elements.

You can use our other components, such as buttons and form groups, inside .ether-toolbar to add functionality to your application.

With Panel

Our framework also includes panels that can be inserted into your layout. Simply add .ether-panel inside of .ether-content-columns before .ether-content.

The panels come with .panel-header, .panel-body, and .panel-footer classes for additional layout options.

You also have the option of including .static-container and .scroll-container inside the .panel-body for additional functionality.

Note: Ether Panels must be given a width using any of the following classes:

  • .panel-sm (250px)
  • .panel-md (350px)
  • .panel-lg (450px)
