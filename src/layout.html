Ether Layout

Besides the aesthetics and UI elements, the Ether Framework has an optional layout you can use for your web applications.

Basic Structure

The page should be wrapped in a <div class="ether-layout"> to set up the overall structure.

The top nav bar is a standard Bootstrap <nav class="navbar navbar-expand-lg"> with an additional class of ether-header to apply the aesthetics.

The navbar should be followed by <div class="ether-body"> to set up the left nav area and main content area.

The left nav is another Bootstrap <nav class="nav flex-column"> to create vertical nav along with an additional class of ether-sidebar for the proper aesthetics. Within each navlink it is important to have and icon or image with a class of icon

The next sibling element after the left nav will automatically become the main content area. The ether-content-area should be closed after this.

After the ether-body the next element should be the <div class="ether-footer">. Each sibling element will be spaced automatically.
