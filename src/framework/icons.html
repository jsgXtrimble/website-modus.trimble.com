Icons

Guidance and suggestions for using external Ether icon library.

Usage Guidelines

The easiest way to start using our icons is to include the following code in the <head> of your page. 

<link rel="stylesheet" href="https://styles.trimblemaps.com/assets/${etherIconsVersion}/fonts/ether-icons.css">

Then simply use <i class="ether-icons>icon_name</i> with the name of the icon inside to insert it into your page.
