Chips

Chips are compact elements that represent an input, attribute, or action.

Code Structure & Usage

Chips are used to add information, make selections or filter content. Chips should appear dynamically as a group of multiple interactive elements.

To use chips, add classes chip

To specify the function of the chip, use chip-input, chip-choice, chip-filter, chip-action

To specify the style, use one of these classes: chip-solid, chip-reverse chip-outline
