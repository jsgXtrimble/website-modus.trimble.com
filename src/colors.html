Theme Colors

These can be changed to define app theme

$theme-colors: (

"primary": #005F9E,

"secondary": #8C8B96,

"dark": #363545,

"success": #5e9331,

"danger": #C81922,

);

UI Colors

These can be changed to define app theme

$ui-colors: (

"main-background": #fff,

"panel-background": #F3F3F7,

"active": #009AD9,

"hover": #EAEAEF,

"selected": #D0D0D7

);
